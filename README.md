I'm a marketer who kept running into the same problem: the tools existed, but the AI layer didn't.

So I learned to build it. Docker, Linux, servers, Node.js, WordPress internals. I didn't set out to be a developer. I just got tired of waiting for someone else to build what I needed.

Three things I actually do well: **marketing, building products, and vibe coding** until something works.

Everything on this page started as something I built for myself. When it saved me enough time, I cleaned it up and shipped it.

Check out everything at **[adityaarsharma.com](https://adityaarsharma.com)**

## What I've Built (and Use Every Day)

| Tool | The Problem | What It Does Now |
|------|-------------|-----------------|
| [youtube-marketing-skills](https://github.com/adityaarsharma/youtube-marketing-skills) | YouTube SEO was eating hours. Every video. | 21 commands, 10 live tools. Reads my channel, writes the SEO, pushes directly to YouTube. Works in Claude Code, Cursor, Codex, Gemini CLI. Replaces VidIQ. |
| [perch](https://github.com/adityaarsharma/perch) | Managing 10+ servers meant living in browser tabs. | 135 RunCloud operations via Claude + Telegram. I message it. It executes. |
| [orbit](https://github.com/adityaarsharma/orbit) | Plugin QA before each release was slow and manual. | 11-check automated suite. Playwright, visual regression, accessibility. One command. |
| [pickle](https://github.com/adityaarsharma/pickle) | ClickUp + Slack together is a context-switching nightmare. | Scans both every morning, tells me what actually needs my attention, drafts the follow-ups. |
| [rankready](https://github.com/adityaarsharma/rankready) | Getting WordPress content cited by AI systems was complicated. | Auto-generates llms.txt, EEAT schema, FAQ, AI summaries, Markdown endpoints. One plugin. |
| [jyotisha](https://adityaarsharma.com) | Vedic astrology calculations needed expensive software. | 28 MCP tools doing live Vedic calculations. Natal charts, dashas, muhurtas, Panchanga. |
