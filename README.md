I'm a marketer who builds things. Over the years, working across WordPress products, YouTube channels, and server infrastructure — I kept hitting the same wall: the tools existed, but the AI layer didn't.

So I built it. One problem at a time.

My three zones: **Marketing · Product Building · Vibe Coding** — and increasingly, all three happen in the same session.

Everything I ship starts from a workflow I actually run. When something saves me hours, I turn it into a tool and open-source it or release it freemium. Not to build a company. To solve the problem properly.
